## 🎌 Landing Page - Chihiro

Este projeto é uma landing page inspirada no universo do filme A Viagem de Chihiro, trazendo uma interface visual encantadora baseada na estética japonesa e no estilo artístico do Studio Ghibli.

A página foi desenvolvida com foco em design moderno, cores suaves e elementos visuais que remetem à jornada da personagem Chihiro Ogino, criando uma experiência imersiva para o usuário.

## 🚀 Funcionalidades

-📱 Design responsivo para diferentes dispositivos
- ✨ Animações e efeitos visuais suaves
- 🧭 Navegação simples e intuitiva

## 🛠️ Tecnologias Utilizadas

- HTML5 🟧
- CSS3 🟦

## 🎯 Objetivo do Projeto

O principal objetivo deste projeto é praticar habilidades de desenvolvimento front-end, explorando conceitos de design, responsividade e criação de interfaces temáticas baseadas em referências culturais marcantes.
